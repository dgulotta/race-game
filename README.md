# race-game

A puzzle game about designing racetracks.

There is a (possibly out of date) web demo [here](https://dgulotta.github.io/race-game-demo/race.html).

To run natively:
```
cargo run --release
```

To run in a web browser:
```
cargo install trunk
trunk serve --release
```
